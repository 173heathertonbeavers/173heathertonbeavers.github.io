Using Cards with Links:

```html
<section class="sections">
  <div class="container">
    <h2>CONTAINER TITLE</h2>
    <div class="cards">
      <info-card 
        title="TITLE" 
        content="CONTENT" 
        link="LINK.html">
      </info-card>
    </div>
  </div>
</section>
```

Later (custom domain):
You just change:
'''<base href="/">'''


'''✅ If using <base href="/173-Heatherton-Scouts/">'''
Then you can simplify to:
'''<link rel="icon" href="favicon.png">'''
✅ If NOT using <base>

Use full path:

<link rel="icon" href="/173-Heatherton-Scouts/favicon.png">
